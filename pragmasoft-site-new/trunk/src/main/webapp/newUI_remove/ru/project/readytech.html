ReadyTech Axis

Технологии: Spring Boot, Hibernate, Redis, Docker, Amazon Web Services, Google Cloud, Microsoft Azure, AngularJS, Websockets, Recurly payment gateway

ReadyTech Axis это SAAS платформа, которая позволяет осуществлять планирование согласованных облачных ресурсов с целью организации учебных мероприятий

  • Поддерживает автоматическое управление облачными ресурсами для крупных поставщиков облачных вычислений
  • Приложение построено с использованием принципов DDD
  • Кластиризированный и сбалансированный по нагрузке бэкэнд позволяет масштабировать и обеспечивать отказоустойчивость
