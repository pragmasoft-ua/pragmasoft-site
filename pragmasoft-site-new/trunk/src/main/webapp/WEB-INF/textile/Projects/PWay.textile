h1. Promotional Way

URL: "http://www.promotionalway.com/":http://www.promotionalway.com/

*Technologies used:* Apache Wicket 1.3.5, MySQL, Tomcat 6.x, KonaKart 2.2.6.0, AJAX, Log4J, JUnit, XHTML, CSS 2.0, JavaScript

Promotional Way specializes in corporate gifts, and can place just about any corporate info that one requires on a range of different products. Pragmasoft was in charge of developing an e-shop for marketing Promotional Way services to a wider range of public.

This project was to make a dynamic, relational database driven web site that used J2EE and servlets. "KonaKart":http://www.konakart.com/ was chosen to be integrated as shopping cart software solution. The entire site (web application + KonaKart) was  implemented on Tomcat standalone. We used the best practices for search engine optimization (SEO) and security.

*E-shop features:*

*(star) product search by name and price range
* recommended products list
* hot products list
* recently viewed products
* recently updated products
* sophisticated order cost calculator (product per-item cost depends on order amount, selected imprinting and shipping)
* payments support with Authorize.net
* custom products attributes
* Konakart admin used as shopping cart backend for order management, product management, shop configuration
* site security with SSL
