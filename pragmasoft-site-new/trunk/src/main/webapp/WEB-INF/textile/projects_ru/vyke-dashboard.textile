h1. Vyke Dashboard


<div class="vc-box">

*Технологии:* JavaEE, JSF, RichFaces, AJAX4JSF, SIFR, Facelets, Blueprints CSS.

Во время работы над Vyke Dashboard, нашей целью было обеспечить интегрированную точку доступа к объектам управления Vyke. Мы использовали преимущества Atlassian Crowd. Crowd это веб версия инструмента единого входа (SSO)  которая направлена на упрощение обслуживания приложений и управления идентификационными данными. Vyke Dashboard состоит из консоли администрирования и интеграции API, где интегрированы такие веб-приложения: JIRA, Confluence и т.д.

Прагмасофт дизайн-студия предоставила стильный дизайн вместе с удобным настраиваемым графическим интерфейсом, а также одним из условий было осуществление HTML/CSS кодирования с технологией BlueprintCSS.
