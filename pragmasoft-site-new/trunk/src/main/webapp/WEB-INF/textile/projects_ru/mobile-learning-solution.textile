h1. Mobile Learning Solution


<div class="vc-box">

*Технологии:* JavaEE, Spring, Hibernate, JavaME, C++, ATL, WTL, MFC.

Mobile Learning Solution - это кроссплатформенная электронная обучающая система для пользователей мобильных телефонов.
Mobile Learning Solution использует инновационную идею применения Системы управления обучением (LMS) на мобильных платформах. Все решение заключается в обучения модуля дизайнера (приложения для настольного ПК), который может построить автономные обучающие приложения с богатым содержанием для ряда мобильных платформ (JavaME, Windows Mobile, Pocket PC, Symbian, WAP-браузерами, веб-браузеров). Кроме того платформа содержит веб-сервер для распространения модулей, развертывание web-/ wap модулей, отслеживания результатов и интеграции с другими обучающими системами (Blackboard, Moodle).

Мы несем ответственность за весь жизненный цикл проекта, начиная с архитектурного проектирования, проведения камеральной дизайнер, зрителей мобильных платформ, веб-сервер, включая документацию, поддержку и обслуживание.
