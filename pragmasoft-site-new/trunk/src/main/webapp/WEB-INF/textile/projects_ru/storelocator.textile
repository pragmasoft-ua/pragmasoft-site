h1. Storelocator


<div class="vc-box">

*Технологии:*  Apache Cordova/PhoneGap, jQuery mobile, Facebook and Google OAuth 2.0, Google Maps API, Geolocation API.

Storelocator теперь доступен для мобильных телефонов, его можно скачать "здесь":http://storelocator.no/.
Наконец Storelocator доступен для мобильного телефона и просто необходим для тех, кто любит ходит по магазинам. Теперь вы можете сэкономить время бегая в незнакомых городах в поисках нужного вам магазина. Простой поиск быстро найдет магазины, где интересующий вас бренд есть в наличие.

Если вы любите путешествовать и ходить по магазинам, то Storelocator для мобильных устройств является основным инструментом для покупок подкованных клиентов. Планируйте свою поездку заранее, создавая онлайн гидов и добавляйте магазин в список желаемых посетить. После того как вы прибыли в пункт назначения, открывайте руководство на вашем смартфоне, находите магазины и осуществляйте покупки прямо во время прогулки.
