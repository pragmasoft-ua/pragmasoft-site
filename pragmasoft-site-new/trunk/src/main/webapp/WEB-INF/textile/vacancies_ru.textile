h2. Java EE разработчик

*Требования к кандидату:*

# Технический английский уровня Intermediate - свободное чтение документации и документирование кода.
# Linux на уровне пользователя.
# Общие знания ООП, паттернов проектирования, метрик качества. Чтение UML - плюс.
# Опыт работы со средствами разработки - maven, junit, SCM (svn/git/mercurial), IDE, debug, profiling, app servers
# Серверная java опыт не менее 3-х лет непрерывной практической работы, включая:
** Язык и платформа Java, JDK tools. наличие сертификата OCP - плюс. 
** Java SDK, включая concurrency, io, nio, networking, collections, classloading, security, reflection, jdbc, annotations, generics
** JavaEE, включая Servlets, JSP, JSTL, EJB3, JPA, JMS, CDI, JAX-RS
** SQL, базы данных, ORM - JPA или Hibernate. NoSQL - плюс
** Знание HTTP (включая заголовки, куки, авторизацию, SSL)
** Frontend технологии (HTML, CSS, JS, Ajax). Frontend MVC фреймворки, SPA, Less/Sass, Grunt/Gulp - плюс
** Spring framework, Spring security
# Опыт фриланса - плюс
