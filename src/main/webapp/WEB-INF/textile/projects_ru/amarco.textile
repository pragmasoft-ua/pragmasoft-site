h1. Amarco


<div class="vc-box">

*Технологии:* JavaEE, Spring, Hibernate, Java Web Start, SWT: The Standard Widget Toolkit, Apache Jackrabbit, PayPal SDK.

Desktop Marketer предназначен для распространения рекламной информации. Он состоит из 2х частей: удобного веб-приложения для создания маркетинговых акций для маркетеров, и специального пользовательского интерфейса для получения рассылки для их подписчиков. Веб-приложение Desktop Marketer является удобным инструментом для составления маркетинговых программ, рассылки информации пользователям и для анализа эффективности завершенных программ.
