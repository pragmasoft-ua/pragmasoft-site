*Технологии:* Adobe Flex, Java, JavaScript, REST, Spring, Hibernate, Apache Click, FreeMaker, Batik, Jersey, jQuery.


Prinly является мощным онлайн инструментом, предназначенный для самостоятельной разработки индивидуальной продукции. Вы можете легко создавать визитные карточки, бланки , конверты и открытки с помощью Prinly Design Tool. Pragmasoft потребовалось значительное усилие, чтобы сделать взаимодействие с Prinly интуитивным , легким и приятным. Prinly Дизайнер оказался современным инструментом построенным с использованием новейших технологий. Теперь вам не придется искать дизайн-студию и осваивать какие либо сложные приложения, вы можете легко создать визитку по своему усмотрению в конструкторе Prinly Online. Это быстро, дешево, и вы получаете прекрасный результат!

Pragmasoft команда работала на Prinly с нуля. В рамках проекта было реализовано:

* Этап проектирования, в том числе дизайн логотипа , дизайн сайта, анализа UI / UX и развития эффективного интерфейса для онлайн дизайнера, HTML / CSS 
* Разработка онлайн - Design Tool с нуля на Flex.
* Разработка на стороне сервера, написание сценариев.

Design Tool заслуживает подробного рассмотрения. Команда Pragmasoft посвятила особое внимание повышению удобства использования для Card Designer, который работает гладко и быстро. С помощью конструктора Prinly Card вы можете создать визитную карточку, используя один из многочисленных стандартных фонов, доступных на Prinly, разработать карту без/с сплошным фоном, или загрузить свой ​​собственный фон в систему. Design Tool позволяет свободно работать с текстами, загружать собственные изображения и фона; все доступные панели расположены в виде аккордеона, который эффективно использует пространство экрана и обеспечивает просторной рабочей зоной.
