h1. Acai Author


<div class="vc-box">

*Технологии:* Sping MVC, Spring Security, Spring Data JPA, Tiles, Bootstrap, JQuery, MarkItUp!, DZSlides.

Веб-приложение для создания/просмотра/прохождения презентаций и тестов для студентов и учеников. Вцелом, приложение расчитано на любую аудиторию: школу, университет, курсы повышения квалификации, и т.д.. Большой набор параметров позволяет создать очень разнообразные модули.

Особенности:
* контент оптимизирован под мобильные устройства
* оффлайн режим (для ранее загруженных модулей)
* разнообразный контент для построения модулей: текст (с различными типами форматирования), изображения, музыкальные файлы.
